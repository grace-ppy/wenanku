# 文案库网站

一个可部署到 GitHub Pages 的纯静态文案库。

## 功能

- 目录分类
- 搜索筛选
- 文案详情页
- 永久链接
- 一键复制标题、文案、全文
- 粘贴完整文本后自动拆解
- 浏览器内新增分类和文案
- JSON 导入 / 导出

## 更新内容

页面内新增的内容会保存到当前浏览器。需要让线上网站长期展示这些内容时：

1. 点击页面左侧「导出 JSON」
2. 用导出的文件替换 `data/copies.json`
3. 提交到 GitHub，GitHub Pages 会自动更新

删除按钮默认隐藏。站点维护者可在自己的浏览器访问 `?admin=1` 开启管理模式，例如：

```text
https://grace-ppy.github.io/wenanku/?admin=1
```

支持的拆解格式：

```text
主题：
分类：
视频链接：
文案：
标签：
```
